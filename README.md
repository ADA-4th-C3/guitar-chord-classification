## Chord Classification


## How to install
1. install python 3.10
```
brew install python@3.10
```

1. create virtual environment
```
python3.10 -m venv .venv
```

1. activate virtual environment
```
source .venv/bin/activate
```

1. install packages
```
pip install -r requirements.txt
```

## Training
```
make training
```

## Run
```
make run
```
